# 模型配置架构设计文档

## 1. 概述

本文档描述了智能知识库系统中模型配置功能的架构设计方案，支持为不同业务场景配置多个优先级的大模型，并实现自动降级策略。

## 2. 业务场景定义

根据项目分析，系统包含以下7个业务场景：

| 场景编码 | 场景名称 | 说明 | 推荐模型类型 |
|---------|---------|------|------------|
| `RAG_CHAT` | RAG智能问答 | 基于知识库文档的智能问答 | 强模型 |
| `QUERY_REWRITE` | 查询改写 | 智能改写用户查询 | 强/中模型 |
| `HYDE` | HyDE假设答案生成 | 生成假设答案增强检索 | 中模型 |
| `ENTITY_EXTRACTION` | 实体提取 | 从文档中提取关键实体 | 轻量模型 |
| `RERANK` | Cross-Encoder重排序 | 对检索结果进行精排 | 中模型 |
| `CONVERSATION_SUMMARY` | 对话摘要生成 | 压缩对话历史 | 轻量模型 |
| `EMBEDDING` | 文档向量化 | 将文档转换为向量表示 | 嵌入模型 |

### 4.3 核心服务类

#### 4.3.1 ModelSelectorService.java（模型选择器）
```java
@Slf4j
@Service
@RequiredArgsConstructor
public class ModelSelectorService {

    private final ScenarioModelConfigRepository configRepository;
    private final ChatModelFactory chatModelFactory;
    private final List<FallbackStrategy> fallbackStrategies;
    private final CacheManager cacheManager;

    /**
     * 为指定业务场景选择模型并执行
     * 
     * @param scenarioCode 业务场景编码
     * @param modelExecutor 模型执行器函数
     * @return 执行结果
     */
    public <T> T executeWithModel(String scenarioCode, Function<ChatModel, T> modelExecutor) {
        List<ScenarioModelConfig> configs = getConfigsForScenario(scenarioCode);
        
        if (configs.isEmpty()) {
            throw new ModelConfigException("场景 " + scenarioCode + " 未配置模型");
        }

        FallbackStrategy strategy = selectFallbackStrategy(scenarioCode);
        return strategy.execute(configs, modelExecutor);
    }

    /**
     * 获取场景的模型配置列表（按优先级排序）
     */
    private List<ScenarioModelConfig> getConfigsForScenario(String scenarioCode) {
        Cache cache = cacheManager.getCache("modelConfigCache");
        String cacheKey = "scenario:" + scenarioCode;
        
        @SuppressWarnings("unchecked")
        List<ScenarioModelConfig> cached = cache.get(cacheKey, List.class);
        if (cached != null) {
            return cached;
        }

        List<ScenarioModelConfig> configs = configRepository
            .findByScenario_ScenarioCodeAndIsEnabledTrueOrderByPriorityAsc(scenarioCode);
        
        cache.put(cacheKey, configs);
        return configs;
    }

    private FallbackStrategy selectFallbackStrategy(String scenarioCode) {
        return fallbackStrategies.stream()
            .filter(s -> s.supports(scenarioCode))
            .findFirst()
            .orElse(new PriorityFallbackStrategy());
    }
}
```

#### 4.3.2 PriorityFallbackStrategy.java（优先级降级策略）
```java
@Slf4j
@Component
public class PriorityFallbackStrategy implements FallbackStrategy {

    @Override
    public boolean supports(String scenarioCode) {
        return true;
    }

    @Override
    public <T> T execute(List<ScenarioModelConfig> configs, Function<ChatModel, T> executor) {
        List<ScenarioModelConfig> enabledConfigs = configs.stream()
            .filter(ScenarioModelConfig::getIsEnabled)
            .toList();

        if (enabledConfigs.isEmpty()) {
            throw new ModelConfigException("没有可用的模型配置");
        }

        Exception lastException = null;
        
        for (ScenarioModelConfig config : enabledConfigs) {
            try {
                return executeWithConfig(config, executor);
            } catch (Exception e) {
                log.warn("模型执行失败，尝试降级: model={}, error={}", 
                    config.getModelInstance().getModelCode(), e.getMessage());
                lastException = e;
                
                if (!config.getFallbackEnabled()) {
                    break;
                }
            }
        }

        throw new ModelConfigException("所有模型均执行失败", lastException);
    }

    private <T> T executeWithConfig(ScenarioModelConfig config, Function<ChatModel, T> executor) {
        ModelInstance modelInstance = config.getModelInstance();
        String providerCode = modelInstance.getProvider().getProviderCode();
        
        ChatModel chatModel = chatModelFactory.getModel(providerCode);
        
        int retries = 0;
        while (retries <= config.getMaxRetries()) {
            try {
                return executeWithTimeout(chatModel, executor, config.getTimeoutMs());
            } catch (Exception e) {
                retries++;
                if (retries > config.getMaxRetries()) {
                    throw e;
                }
                log.warn("模型调用重试: model={}, retry={}/{}", 
                    modelInstance.getModelCode(), retries, config.getMaxRetries());
            }
        }
        throw new IllegalStateException("不应到达此处");
    }

    private <T> T executeWithTimeout(ChatModel model, Function<ChatModel, T> executor, int timeoutMs) {
        CompletableFuture<T> future = CompletableFuture.supplyAsync(() -> executor.apply(model));
        try {
            return future.get(timeoutMs, TimeUnit.MILLISECONDS);
        } catch (TimeoutException e) {
            future.cancel(true);
            throw new ModelConfigException("模型调用超时");
        } catch (Exception e) {
            throw new ModelConfigException("模型调用失败", e);
        }
    }
}
```

### 4.4 控制器设计

#### ModelConfigController.java
```java
@Tag(name = "模型配置管理", description = "业务场景模型配置管理接口")
@Slf4j
@RestController
@RequestMapping("/api/model-config")
@RequiredArgsConstructor
public class ModelConfigController {

    private final BusinessScenarioService scenarioService;
    private final ModelProviderService providerService;
    private final ModelInstanceService instanceService;
    private final ScenarioModelConfigService configService;

    @GetMapping("/scenarios")
    public ApiResponse<List<BusinessScenarioDTO>> listScenarios() {
        return ApiResponse.success(scenarioService.listAll());
    }

    @GetMapping("/scenarios/{scenarioCode}/config")
    public ApiResponse<ScenarioConfigDetailDTO> getScenarioConfig(@PathVariable String scenarioCode) {
        return ApiResponse.success(configService.getConfigDetail(scenarioCode));
    }

    @PutMapping("/scenarios/{scenarioCode}/config")
    public ApiResponse<Void> updateScenarioConfig(
            @PathVariable String scenarioCode,
            @Valid @RequestBody UpdateScenarioConfigRequest request) {
        configService.updateConfig(scenarioCode, request);
        return ApiResponse.success();
    }

    @GetMapping("/providers")
    public ApiResponse<List<ModelProviderDTO>> listProviders() {
        return ApiResponse.success(providerService.listAll());
    }

    @PostMapping("/instances")
    public ApiResponse<ModelInstanceDTO> createModelInstance(
            @Valid @RequestBody CreateModelInstanceRequest request) {
        return ApiResponse.success(instanceService.create(request));
    }

    @GetMapping("/instances")
    public ApiResponse<List<ModelInstanceDTO>> listModelInstances(
            @RequestParam(required = false) String providerId,
            @RequestParam(required = false) ModelType modelType) {
        return ApiResponse.success(instanceService.list(providerId, modelType));
    }
}
```
